**A Full-Stack Mern application where users can :** <br>
-> Login/register <br>
-> Create a post <br>
-> Like a post (only when he is signed-up) <br>
-> Delete his own post <br>

**Future Improvements :** <br>
-> open a post to read the full article <br>
-> related posts <br>
-> comment on a particular post

**Tech Stack :** <br>
-> React-Redux <br>
-> MongoDB <br>
-> Node.js <br>
->Material-UI
